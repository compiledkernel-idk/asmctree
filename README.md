# 3D Christmas Tree

Animated 3D Christmas tree for Wayland.

## Dependencies

```
wayland-client wayland-protocols gcc make
```

Arch: `pacman -S wayland wayland-protocols`

## Build & Run

```bash
make
./christmas_tree
```

## Features

- Animated falling snow
- Twinkling lights
- 3D shaded ornaments
- Glowing star
- Night sky with stars
